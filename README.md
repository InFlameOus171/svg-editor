# \<svg-editor>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Install required packages

```bash
npm install
```

## Local Demo with `web-dev-server`

```bash
npm start
```

## Linting

```bash
npm lint
```

## Building

```bash
npm run build
```

## Usage

```html
<script type="module">
  import 'svg-editor/svg-editor.js';
</script>

<svg-editor></svg-editor>
```
